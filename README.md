# Watch My Budget

This is a project generated with Angular CLI 1.5. It is a simple app for tracking budgets for events.

![](doc/screenshot.png?raw=true)

## Technology Used

- Angular 5
- MongoDB Stitch (uses v1.0 of the API)
- Angular Material
